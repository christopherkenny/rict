# rict: Redistricting in Clean Tables

Provides a suite of tools to create tables that accompany maps. The
tools create clean, informative tables for electoral outcomes,
compactness, and other district-level quantities. Most tools are aimed
at the redistricting context, but are broadly applicable to other
electoral data.
